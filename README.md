# Proyecto base para la comision 999200928
## Maquetacion de proyecto web - Clon Whatsapp

Este proyecto se centra en el desarrollo visual de una interfaz de chat similar a Whatsapp. Para llevarlo a cabo se utilizo HTML y CSS. Incluye una sidebar con el perfil del usuario y una lista de chats, un panel principal con una coversación y un footer con un textarea para enviar mensajes.

## Decisiones de diseño:
1. Inclusión de un ícono de lupa en la barra de búsqueda.Para esto se utilizó position, ya que esta propiedad permite posicionar un elemento sobre otro. En específico se aplicó position: relative en el contenedor .search y position: absolute al ícono, lo que permitió posicionar la lupa dentro del input search. 
Con las propiedades top y left se ajustó la ubicación del ícono para centrarlo en la posición deseada.
2. Incorporación de imágenes tanto para los perfiles de los usuarios como para el fondo del panel de chat, para mejorar la apariencia visual.
3. Se agregaron bordes en la parte superior e inferiro de la barra de búsqueda.
4. Inclusión ícono whatsapp.
