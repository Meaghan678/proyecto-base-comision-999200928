# Proyecto base para la comision 999200928
## Maquetacion de proyecto web - Clon Whatsapp

Este proyecto se centra en el desarrollo visual de una interfaz de chat similar a Whatsapp. Para llevarlo a cabo se utilizo HTML y CSS. Incluye una sidebar con el perfil del usuario y una lista de chats, un panel principal con una coversación y un footer con un textarea para enviar mensajes.

└── foto-perfil-3
└── fondo-2

## Cómo visualizar el proyecto
Se debe abrir el archivo "index.html" en cualquier navegador web.
